Scripting Command Reference

add: Add an object:
autosize: Resize a object regardless of zoom
center: Place a point in the center of the screen
fastrewind: Reset the score to the start
play: Play the score
registerColor: Define and name a custom color
registerColor2: Documentation To Do
registerTexture: Define and name a custom texture
remove: Delete an object
rotate: Change the viewing direction
setActive: Enable or disable an object
setBoundsSource: Indicate the interval in which your cursor is located (for mapping):
setBoundsTarget: Set the range of cursor values to send (mapped from above definition of bounds source)
setColorActive: Color of an active object that does not send any messages
setColorActive2: Documentation To Do
setColorActiveMessage: Color of an active object that sends messages
setColorActiveMessage2: Documentation To Do
setColorInactive: Color of an inactive object that does not send any messages
setColorInactive2: Documentation To Do
setColorInactiveMessage: Color of an inactive object that sends messages
setColorInactiveMessage2: Documentation To Do
setCurve: Assign a cursor to a curve
setGroup: Set the group of an object
setLabel: Set the label of an object
setLine: Documentation To Do
setMessage: Set the message of an object
setOffset: Set the starting point offset of a cursor on its curve
setPattern: Set the direction and number of cycles of a cursor
setPointAt: Add or change a point on a line segment of a path
setPointsEllipse: Add a circle or ellipse
setPointsImg: Documentation To Do
setPointsSVG: Define all the points of an SVG (Scalable Vector Graphics) cubic Bezier path
setPointsSVG2: Define all the points of an SVG (Scalable Vector Graphics) line-segment path
setPointsTxt: Display a curve shaped as a character
setPos: Set the position of an object
setResize Change the size of an object
setSize: Change the thickness of a curve or cursor
setSpeed: Set the speed of a cursor in proportion to the master transport speed
setSpeedF: Control the instantaneous speed of a cursor
setTextureActive: Set the texture of an active object that does not send any messages
setTextureActiveMessage: Set the texture of an active object that sends messages
setTextureInactive Set the texture of an inactive object that does not send any messages
setTextureInactiveMessage: Set the texture of an inactive object that sends messages
setTime: Set the instantaneous position of a cursor
setTriggerOff: Set the time before release (NoteOff) for a trigger
setWidth: Set the width of the play head (cursor)
speed: Set the global playing speed of the score
zoom: Set the visible display area
