Raccourcis

Partition et objets :

  • CONTROL/COMMAND : naviguer dans la partition sans déplacer ou sélectionner les objets
  • SHIFT : ajouter des objets dans la sélection
  • ALT : navigation 3D
  • WHEEL : zoom
  • SHIFT+WHEEL : zoom rapide
  • ESC : annule la sélection ou le plein écran

Édition de courbe:

  • DOUBLE-CLICK sur la courbe : ajouter un point sur la courbe
  • DOUBLE-CLICK sur un point : bascule entre Bezier ou ligne droite
  • CONTROL/COMMAND:+DOUBLE-CLIC supprimer un point sur la courbe

Menus de contexte

Un menu de contexte est fourni pour manager les fichiers dans la liste "Scripts & Style" de l'inspecteur. Le menu s'ouvre avec un clic droit dans un script ou un fichier de style. (Ou utiliser ctrl+clic sur Macintosh).

Commandes du menu Script-File

  • Run: Exécute le script ou le style dans le fichier
  • New: Créer un nouveau script ou fichier de style
  • Duplicate: Duplique le script ou le fichier de style
  • Delete: Supprime le script ou le fichier de style
  • Open: Ouvre le script ou le fichier de style
  • Open in external editor: Ouvre le script ou le fichier de style dans l'éditeur de système enregistré pour les fichiers *.nxscript.

Note: Un fichier de style est similaire à un script mais doit seulement contenir des commandes qui altèrent l'apparence de la partition.
